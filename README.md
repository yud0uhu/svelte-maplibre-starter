# Usage

<img src="static/README02.png" width="300" alt="README">

<br>

Install package

```bash
pnpm install
```

<br>

<br>

code format

```bash
pnpm run format
```

<br>

build

```bash
pnpm run build
```

<br>

dev

```bash
pnpm run dev
```

<br>

Unit Tests

```sh
pnpm run test:unit
```

<br>

Lint

```sh
pnpm run lint
```
